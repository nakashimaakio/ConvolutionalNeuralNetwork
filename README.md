# ConvolutionalNeuralNetwork

##事前準備

visual c++ 2010で実装

* opencv2.2をあらかじめ追加  
* プロパティ→C/C++→追加のインクルードディレクトリ：C:\OpenCV2.2\include  
* プロパティ→全般→文字セット：Unicode 文字セットを使用する  

##使い方

使い方その１〜４まであるので、自分の好きなやり方で実装してみてください．

以下にそれぞれの特徴をまとめた表を載せておきます．

|使い方|その１|その２|
|難易度|初心者|中級者|
|コマンドライン引数|なし|あり|



###使い方その１

初心者向け、コマンドライン引数なし；１つずつ画像を学習したいときに最適

１：画像を選択→モード１（初期化）の選択

２：画像を選択→モード３（学習）の選択→教師信号の選択　（何回か違う画像で繰り返す）  
※教師信号はあらかじめ自分で数字を割り振っておく

３：画像を選択→モード２（識別）の選択
